# Mission Control by NotifiUs, LLC

Tools and services for the NAEO community and Amtelco ecosystem. 

## Overview

Mission Control is a 64-bit Windows application which contains a variety of tools and services (free and paid) that augment your Amtelco-based environment. 

## Download

You can download the latest release at [github.com/CallTheory/mission-control/releases](https://github.com/CallTheory/mission-control/releases)

Select the version you wish to install. From here, you have two options:

* Download the `.zip` file and extract the contents.
* Download the `mission-control-setup-x.x.x.exe` file directly

## Install

Run `mission-control-setup-X.Y.Z.exe` to install the application (downloaded directly, or in the extracted folder.)

## Keeping up to date

The application has built in auto-update functionality which will install the latest version when you exit. 

You won't need to regularly perform manual downloads for the latest version.  

## Other files

If you download the `.zip` file from the release page, there are some additional files included. 

This file is for version control and can safely be ignored: 

* `.gitignore` - This file tells version control what to ignore. 

These files are for auto-update functionality and can safely be ignored:

* `latesty.yml` - This file includes information on the latest version, including checksum hashes. 
* `mission-control-setup-X.Y.Z.exe.blockmap` - This file includes mapping information for application updates.
