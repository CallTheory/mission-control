# Mission Control

Tools and services for the NAEO community and Amtelco ecosystem for [Call Theory](https://calltheory.com) subscribers and the NAEO community.

## Overview

Mission Control is a 64-bit Windows application which contains a variety of tools and services (free and paid) that augment your Amtelco-based environment. 

## Install

You can get the latest release here: [github.com/CallTheory/mission-control/releases](https://github.com/CallTheory/mission-control/releases)

After downloading the `mission-control-setup-x.x.x.exe` setup file - run it to install the application.


## Keeping up to date

The application has built in auto-update functionality which will install the latest version when you exit. 

You won't need to regularly perform manual downloads for the latest version.
